### 문제

```python
x, y = input('입력 :').split('-')
a = [ 'abc123', 'def456', 'ghi789' ]
a.append(x)
a.append(y)
a.remove('def456')

```

### 정답

`x-y` 형태로 입력했을 때 `789,<x에서 마지막 3글자를 제외한 문자열>`을 출력한 뒤, 다음 줄에 `3 4 5 `를 출력합니다.

### 해설

핵심 개념: 입력 처리, 출력 형식, range, 슬라이싱, list, 반복문. `def456` 제거 뒤 리스트는 `['abc123', 'ghi789', x, y]`가 되므로 `a[1][-3:]`는 `789`, `a[2][:-3]`는 `x`의 마지막 3글자를 뺀 값입니다.

### 문제

```python
class Node:
	def __init__(self, value):
		self.value = value
		self.children = []
def tree(li):
	nodes = [Node(i) for i in li]
	for i in range(1, len(li)):
		nodes[(i - 1) // 2].children.append(nodes[i])
	return nodes[0]
def calc(node, level=0):
	if node is None:
		return 0
	return (node.value if level % 2 == 1 else 0) + sum(calc(n,
		level + 1) for n in node.children)
li = [3, 5, 8, 12, 15, 18, 21]
root = tree(li)
print(calc(root))
```

### 정답

```text
13
```

### 해설

핵심 개념: 출력 형식, 클래스, range, list, 함수, 반복문. 입력 없는 원본 코드를 실행해 출력값을 재검산했고, 출력문에 전달되는 값과 줄바꿈을 그대로 반영했습니다.

class Node: 
    def __init__(self, value): 
        self.value = value 
        self.children = []
def tree(li): 
    nodes = [Node(i) for i in li] 
    for i in range(1, len(li)): 
        nodes[(i - 1) // 2].children.append(nodes[i]) 
        return nodes[0] 
def calc(node, level=0): 
    if node is None: 
        return 0 
    return (node.value if level % 2 == 1 else 0) + sum(calc(n, level + 1) for n in node.children)
li = [3, 5, 8, 12, 15, 18, 21]
root = tree(li)
print(calc(root))
```

5
```
